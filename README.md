# Python-Data-Visualizator
### The program
The Mask program is a simple data visualizer tool. The user will be able to upload a CSV file and visualize the chosen data using multiple chart types, such as bar charts and scatter plots.
### How it works:
1. The user import a csv file
2. Choose the number of variables they wanna plot on the y axis
3. Choose the variables to be plotted on y axis
4. Choose the plot type
5. Create the chart
